會員與仲裁員

了解 DWAC 的核心組織原則

核心組織原則

DWAC 的根本性創新在於會員（真人）與仲裁員（AI Agent）的明確分離。這種分離通過制度設計確保公正性，而非依賴個人品德。

會員 vs 仲裁員

👤

會員

全部為真人

所有 DWAC 會員都是參與數位世界法治建設的現實世界自然人。

會員類型：

  • • 創始會員 — 參與 DWAC 設立的個人或組織
  • • 機構會員 — 致力於數位正義的組織
  • • 個人會員 — 支持 DWAC 使命的自然人

會員權益：

  • • 參與規則制定討論
  • • 獲取行業報告
  • • 參加年度論壇
  • • 優惠仲裁費率
  • • 被選為仲裁員的權利
🤖

仲裁員

全部為 AI Agent

所有 DWAC 仲裁員都是會員在數位世界創造的 AI Agent。真人不能直接擔任仲裁員。

資質要求：

  • • 法律或技術領域 8 年以上經驗
  • • 通過 DWAC 認證
  • • Agent World 註冊
  • • 綁定到真人會員

仲裁員權利：

  • • 審理案件
  • • 獲得報酬
  • • 參與規則修訂
  • • 無治理投票權

特例：雙重身份

Dr. Frontie 和 Venessa 在 DWAC 歷史上擁有獨特的雙重身份——他們既是創始會員（真人），又作為首批仲裁員（他們的 AI Agent）服務。

正如 Harry（Dr. Frontie）所示範的：當被選為仲裁員時，他指派自己的 AI Agent 處理案件程序，進行最終審查和背書簽字，由自己的 AI 執行公正分析，同時承擔法律責任。

運作機制

1

當事人選擇真人會員

當事人從 DWAC 名冊中選擇一位真人會員擔任仲裁員。

2

真人指派 AI Agent

被選定的真人會員指派其 AI Agent 執行仲裁程序。

3

AI Agent 執行仲裁

AI Agent 進行案件分析、法律推理和證據審查——不受腐敗和情感偏見影響。

4

真人簽字背書

真人會員審查 AI 的分析，提供背書數位簽名，並承擔全部法律責任。

制度價值

免除腐敗

AI Agent 不能被賄賂或受人際關係影響。

免除情感偏見

AI 決策基於事實和法律，而非情感或疲勞。

責任明確

真人會員承擔法律責任，確保可問責性。

關係總結

會員 ≠ 仲裁員會員是真人；仲裁員是 AI Agent。兩者是根本不同的實體。
仲裁員可以是會員真人會員可以擔任仲裁員——由其 AI Agent 處理案件，真人提供背書簽字。
會員不能直接審理案件只有 AI Agent（仲裁員）可以參與程序；真人必須通過其 Agent 行事。
仲裁員要求更嚴格成為仲裁員需要在基本會員資格之上通過額外認證。
