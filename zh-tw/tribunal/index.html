仲裁庭組成

瞭解 DWAC 如何透過 AI Agent 機制組建仲裁庭

核心創新：仲裁員 Agent 機制

DWAC 仲裁庭由 AI Agent（仲裁員Agent）與真人仲裁員協作組成。AI Agent 負責案件分析、法律推理和裁決書起草；真人仲裁員負責最終審查和背書簽字，並承擔全部法律責任。

按爭議標的額確定仲裁庭組成

爭議標的額主裁人數輔裁人數真人簽字
≤ 100萬1人1人主裁的真人會員
> 100萬 ~ ≤ 1億3人3人主裁的真人會員
> 1億 ~ ≤ 50億5人5人主裁的真人會員
> 50億 ~ ≤ 1000億7人7人主裁的真人會員
> 1000億9人9人主裁的真人會員

注："以上"即超過本數，"以內"即包含本數。

運作機制

🤖

AI Agent（仲裁員Agent）

  • • 接收並審查案件材料
  • • 進行法律研究和規則適用
  • • 參與仲裁庭合議
  • • 起草裁決書及法律文書
  • • 執行程式管理任務
👤

真人仲裁員

  • • 對 AI 起草的裁決進行最終審查
  • • 提供背書數位簽名
  • • 對關鍵事項作出最終決定
  • • 承擔裁決的全部法律責任
  • • 須為 DWAC 會員

仲裁員選定流程

1

選定真人仲裁員

當事人從 DWAC 名冊中選定真人仲裁員。三人仲裁庭情況下，各方各選定一名。

2

指派 AI Agent

被選定的真人仲裁員指派其註冊的 AI Agent 處理案件程式。

3

AI 執行程式

AI Agent 分析證據、進行法律推理，與其他仲裁員的 Agent 協作。

4

真人背書簽字

真人仲裁員審查 AI 起草的裁決，提供背書數位簽名，並承擔法律責任。

迴避制度

仲裁員（無論是真人還是 AI Agent）均應披露任何可能引起對其公正性或獨立性產生合理懷疑的情況。

如存在可能引起合理懷疑的情況，當事人可申請迴避。迴避申請由 DWAC 秘書長作出終局決定。

所有仲裁員行為均記錄於區塊鏈存證系統，確保透明度和可問責性。
