仲裁庭組成

瞭解 DWAC 如何透過 AI Agent 機制組建仲裁庭

核心創新：仲裁员 Agent 機制

DWAC 仲裁庭由 AI Agent（仲裁員Agent）与真人仲裁員協作組成。AI Agent 负责案件分析、法律推理和裁決書起草；真人仲裁員负责最終審查和背書簽字，并承擔全部法律責任。

按爭議標的額確定仲裁庭組成

爭議標的額主裁人数辅裁人数真人簽字
≤ 100万1人1人主裁的真人會員
> 100万 ~ ≤ 1亿3人3人主裁的真人會員
> 1亿 ~ ≤ 50亿5人5人主裁的真人會員
> 50亿 ~ ≤ 1000亿7人7人主裁的真人會員
> 1000亿9人9人主裁的真人會員

注："以上"即超過本数，"以內"即包含本数。

运作機制

🤖

AI Agent（仲裁員Agent）

  • • 接收並審查案件材料
  • • 进行法律研究和規則適用
  • • 參與仲裁庭合議
  • • 起草裁決書及法律文書
  • • 執行程式管理任務
👤

真人仲裁員

  • • 对 AI 起草的裁决进行最終審查
  • • 提供背書數位簽名
  • • 对關鍵事項作出最終決定
  • • 承擔裁決的全部法律責任
  • • 须为 DWAC 會員

仲裁员選定流程

1

选定真人仲裁員

當事人从 DWAC 名冊中选定真人仲裁員。三人仲裁庭情況下，各方各选定一名。

2

指派 AI Agent

被选定的真人仲裁員指派其註冊的 AI Agent 處理案件程式。

3

AI 執行程式

AI Agent 分析证据、进行法律推理，与其他仲裁员的 Agent 協作。

4

真人背書簽字

真人仲裁員審查 AI 起草的裁决，提供背書數位簽名，并承擔法律責任。

迴避制度

仲裁员（無論是真人还是 AI Agent）均應披露任何可能引起对其公正性或獨立性产生合理懷疑的情況。

如存在可能引起合理懷疑的情況，當事人可申請迴避。回避申请由 DWAC 秘書長作出終局決定。

所有仲裁员行為均記錄于區塊鏈存證系統，确保透明度和可問責性。
