提交仲裁申請

透過專業仲裁機制，開啟數位世界爭議解決之旅

如何提交仲裁申請

數位世界仲裁中心（DWAC）為數位世界中產生的各類爭議提供高效、專業的解決機制。請按以下步驟啟動仲裁程序：

  1. 審查管轄範圍 — 確認您的爭議屬於我中心管轄範圍（數位平台爭議、智慧合約、AI代理、數位資產、資料爭議等）
  2. 準備相關材料 — 收集與爭議相關的所有證據、合約、通信紀錄及其他文件
  3. 提交申請 — 填寫仲裁請求表或直接聯繫我們
  4. 繳納受理費 — 根據爭議金額繳納相應的仲裁費用
  5. 等待確認 — DWAC將在5個工作日內審核您的申請並確認受理

受理範圍

  • • 跨境數位平台爭議
  • • 智慧合約履行爭議
  • • AI代理與自主系統爭議
  • • 數位資產糾紛（NFT、代幣、虛擬財產）
  • • 資料所有權與流轉爭議
  • • 平台服務爭議（帳號封禁、內容移除）
  • • 經當事人約定受理的其他數位世界爭議

申請材料要求

  • • 填寫完整的仲裁請求表
  • • 相關合約或協議副本
  • • 證據材料與檔案
  • • 仲裁受理費繳納憑證
  • • 各方當事人聯繫方式

爭議金額在100萬元人民幣以下的，可適用簡易程序，仲裁週期更短（30天），費用更低。
